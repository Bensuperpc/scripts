# New Features !

  - FFMPEG docker scripts
  - Install and Uninstall scripts

#### Install
You need Linux distribution like Ubuntu or Manjaoro

```sh
git clone https://github.com/Bensuperpc/scripts.git
```
```sh
cd scripts
```

```sh
./install.sh
```
#### Uninstall
```sh
./uninstall.sh
```
